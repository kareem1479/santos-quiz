SANTOS QUIZ - نسخة عربية (Front-end only)
===============================

ملف يحتوي واجهة موقع "سانتوس كويز" جاهزة للتشغيل محليًا.
الملفات:
- index.html
- styles.css
- app.js

كيفية التشغيل:
1. فك ضغط الملف (santos_quiz.zip).
2. افتح index.html في المتصفح (دون الحاجة لخادم).
3. الواجهة تعمل محليًا وتستخدم localStorage لمحاكاة تخزين المستخدم والجلسة.
4. هذه نسخة front-end فقط؛ إذا تريد دفع حقيقي أو تسجيل مستخدمين في قاعدة بيانات، أقدر أضيف back-end (Node/Express أو Firebase).

أرغب أن أجهز لك:
- رفع الموقع على استضافة مجانية (Netlify/Vercel) + خطوات بسيطة.
- أو تركيب back-end + بوابة دفع تجريبية (Stripe أو PayTabs) — قلّي أي خيار تفضّل.
